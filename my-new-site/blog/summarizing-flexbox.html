What is Flexbox?

Flexbox is a layout method for designing webpages. It allows for flexible responsive pages without having to mess with float.

How does it work?

A parent element is created and given one or more child elements. 

		
<div class="flex-container">
<div>1</div>
<div>2</div>
<div>3</div>
</div>
To make it flexible, we add a bit of CSS to control the parent container. 
		
			.flex-container {
				display: flex;
			}
From there, we can make them stack in various ways. Such as vertically. 
		
			.flex-container {
				display: flex;
				flex-direction: column;
			}

Question 1

What is a doc root relative link/path?

a path that starts with a slash to represent the root folder, listing the rest of the path after it

Question 2

What character does a doc root relative link/path start with?

/

Question 3

What is the document root directory of a web server?

the folder that the website lives in

Question 4

What is a static site generator?

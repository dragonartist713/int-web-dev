Int Web Dev - Week 9 Questions and Coding Problems

Question 1

What is inheritance in object-oriented programming?

one class (child) gaining the attributes and methods of another class (parent)

Question 2

Why is inheritance a powerful feature of object-oriented programming?

it lets us reuse the code from the parent class for another without rewriting it all again

Question 3

Consider this code sample: 


const d1 = new Dog("Buster", "German Shepard");	
const d2 = new Dog("Buster", "German Shepard");

if(d1 == d2){
	console.log("It's true!");
}else{
	console.log("It's false!");
}

Explain why we should see "It's false!" in the console log.

because d1 and d2 are different objects, despite having the same contents

Question 4

Consider this code sample: 


const d1 = new Dog("Buster", "German Shepard");	
const d2 = d1;

if(d1 == d2){
	console.log("It's true!");
}else{
	console.log("It's false!");
}

Explain why we should see "It's true!" in the console log.

d2 has been assigned the object d1 and so now reference the exact same object, thus they are equal

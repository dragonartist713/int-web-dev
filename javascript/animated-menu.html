Animated Flyout Menu with JavaScript and CSS

Part 1 - Review the CSS Code

Review each of the rulesets in the CSS code for this page. Experiment with them in the developer tools (for example toggle some of the rules by checking the boxes in the CSS developer tools).

Some things to note:

  • Then 'hamburger' icon in the menu button is created by using an entity character.
  • The width of the menu is initially set to 0px, which hides it from the page.
  • If the class attribute of the menu includes open, then the width of the menu changes to 200px, which makes it visible on the page.
  • There is a CSS transition applied to the width property of the menu, which states that if the width of the menu changes, the transition should occur in 1 second.

Make sure to use the developer tools to manually add and remove the 'open' class to the menu. To do this, click on the 'elements' button in the developer tools and then double-click on the menu element to update it's attributes.

Part 2 - Add JavaScript that Adds and Removes the 'open' class on the menu

The JavaScript code that we'll add will listen for clicks on the menu button (not the menu). When the button is clicked we'll toggle the 'open' class on the menu. This will show or hide the menu, and we'll see the CSS transition at work as the width changes.
